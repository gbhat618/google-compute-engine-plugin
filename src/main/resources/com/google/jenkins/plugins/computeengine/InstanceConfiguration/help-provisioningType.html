GCP offers three VM provisioning options, each with different availability, pricing, and termination policies.
  • Standard: Stable provisioning model with guaranteed availability by GCP SLAs. Higher pricing compared to Spot and Preemptible VMs. Use the "Max Run Duration Seconds" field to limit the instance duration and enable automatic deletion.
  • Spot: Low-cost instances that may be terminated if resources are needed elsewhere. Availability varies. Unlike Preemptible VMs, they are not always terminated after 24 hours. Spot and Preemptible VMs are priced the same. Use the "Max Run Duration Seconds" field to ensure termination after a specific duration.

    See Spot VMs for more details.

  • Preemptible: Low-cost instances that may be terminated if resources are needed for other tasks. Availability varies. Spot and Preemptible VMs are priced the same. The "Max Run Duration Seconds" field does not apply to Preemptible instances.

    See the Preemptible VM Instances documentation for more information.
